작성일: 2026-06-08 KST

## 결론

현재 제출 핵심은 Ennoia Studio 앱 `여행 상황 점검`이다. 외부 웹페이지 `https://pockets-merely-brake-trainer.trycloudflare.com`은 선택 제출 또는 보조 시연용 확장 화면으로 둔다.

제출서의 KTO OpenAPI 활용 항목은 현재 기준 `국문 관광정보 서비스`만 선택하는 것이 맞다. Ennoia 앱에 실제 연결되어 확인한 KTO 엔드포인트는 모두 `KorService2` 계열이다.

## 확인 근거

- Notion 참가자 퀵 가이드와 FAQ: Ennoia 앱 공유 URL, 서비스 소개서, KTO OpenAPI 신청자명, 개인 서비스키, 실제 사용 OpenAPI 목록 제출 필요.

- Ennoia Studio 확인: 앱 `여행 상황 점검`, 에이전트 `여행 상황 점검 에이전트`, 모델 `gpt-4o`, 최대 토큰 `2048`, 온도 `0.2`.

- 로컬 직접 호출 확인: KTO `화성행궁` 검색 성공, Kakao 주변 주차장 검색 성공.
- Ennoia 제출 앱 채팅 확인: `6월 23일부터 25일까지 부산...` 추천 질문에서 `[일정 검토]` 응답, `광안리 M(Marvelous) 드론 라이트쇼`, 해운대/기장/남파랑길 등 KTO 행사·관광정보 후보 확인.
- 공개 웹 확인: rpi의 `cloudflared`가 `http://127.0.0.1:8787`을 `https://pockets-merely-brake-trainer.trycloudflare.com`으로 연결 중.
- Ennoia 공식 도움말 확인: Studio 앱 목록의 `... > 공유하기`에서 공유 링크를 생성해야 프로젝트 외부 사용자가 접근할 수 있다. Public 링크는 로그인 없이 접근 가능하고, Restricted 링크는 로그인 및 프로젝트 멤버 권한이 필요하다.
